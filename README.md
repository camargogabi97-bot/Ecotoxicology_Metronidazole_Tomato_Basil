# Ecotoxicology_Metronidazole_Tomato_Basil
This repository contains the processed data and supporting materials used in the scientific article: "Ecotoxicological Effects and Metabolic Fate of Metronidazole in Soil-Plant Systems: Evidence from _Ocimum basilicum_ and _Lycopersicum esculentum_". The goal is to make the research more transparent and its results more reproducible.

The data and files presented here support four main parts of the study:

    1.Germination Test Maps and Germination Rates.

    2.Analysis of Thin-Layer Chromatography (TLC) Plates.

    3.Densitometric Analysis of TLC Images.

    4. Ultra-High Performance Liquid Chromatography coupled with Diode Array Detection and Tandem Mass Spectrometry

  ---

📂 Folder Structure and Contents

/Germination/

- Contains germination test data and calculations, including the crop germination map and corresponding germination rates.

/Thin-layer chromatography/

- Files related to TLC analysis. This includes high-resolution digital images of chromatography plates.

- The detailed protocol for preparing the plates is also available in this folder.


/Densitometric analysis/

- This file contains the chromatographic plates used for quantification, along with the densitometric results of the extracts. It also includes the plate used for the analytical curve and the corresponding graph.


/Ultra-high performance liquid chromatography coupled with diode array detection and tandem mass spectrometry/

- This folder contains the chromatogram images as well as the mass spectra.

- The protocol used to perform the analysis.

OBS: Due to file size limitations, the chromatograms, mass spectra, and analytical protocol have been compressed into a single ZIP archive. The folder contains all raw data supporting the identification of the detected compounds.
